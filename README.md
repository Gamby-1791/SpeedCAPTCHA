# SpeedCAPTCHA
User script that enhances the efficiency of resolving Google reCAPTCHA challenges by reducing the duration of transition effects and enabling uninterrupted selection capability.

![Demonstration](https://github.com/Gamby-1791/SpeedCAPTCHA/blob/main/speedCAPTCHA.gif)
